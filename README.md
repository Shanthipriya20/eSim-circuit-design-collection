# 🔷 eSim Circuit Design Collection

This repository showcases a collection of fundamental digital circuit and IC designs implemented using eSim. It includes schematic designs, simulation files, waveform analysis, and detailed reports for each project.

These projects reflect my hands-on learning in Digital Electronics and Circuit Design, forming a strong foundation for advanced work in FPGA, VLSI, and embedded systems.

## 🛠 Tools Used
- eSim
- Ngspice
- KiCad
